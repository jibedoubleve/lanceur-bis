# Mots-clés, macros et cartes de remplacement

## Mots-clés

Lanceur inclut des mots-clés prédéfinis intégrés qui ne peuvent pas être supprimés ni reconfigurés.

| Mot-clé   | Description                                              | Paramètre                        |
| --------- | -------------------------------------------------------- | -------------------------------- |
| `add`     | Crée un nouvel alias.                                    | Le nom de l'alias à créer.       |
| `setup`   | Ouvre la page des paramètres.                            | N/A                              |
| `centre`  | Centre la barre de recherche sur l'écran.                | N/A                              |
| `version` | Affiche la version de Lanceur.                           | N/A                              |
| `stat`    | Ouvre la page des analytiques.                           | N/A                              |
| `help`    | Ouvre la page d'aide en ligne.                           | N/A                              |
| `logs`    | Ouvre l'emplacement où les journaux sont stockés.        | N/A                              |

## Macros

Les macros sont des mots-clés cachés qui peuvent être activés en créant un alias avec un texte spécifique dans le champ _Nom du fichier_.

Pour activer une macro :

1. Entrez son nom (avec `@`) dans le champ _Nom du fichier_.
2. Attribuez-lui un nom.
3. Enregistrez l'alias.

| Macro                 | Description                                                              | Paramètres                   |
| --------------------- | ------------------------------------------------------------------------ | ---------------------------- |
| `@multi@`             | Exécute une liste d'alias.                                               | La série d'alias             |
| `@guid@`              | Génère un GUID et le copie dans le presse-papiers.                       | N/A                          |
| `@clipboard_as_text@` | Supprime la mise en forme et copie uniquement le texte brut.             | N/A                          |
| `@github_issue@`      | Crée un nouveau ticket GitHub pour Lanceur.                              | Le titre du ticket           |

### `@multi@` - Exécuter une liste d'alias dans un ordre personnalisé

Exécute une liste d'alias dans l'ordre que vous avez configuré. Dans les paramètres, ajoutez tous les alias que vous souhaitez lancer et dans l'ordre souhaité.

Chaque alias est séparé par au moins un `@`. Chaque `@` retarde l'exécution d'une seconde (pour un délai plus long, ajoutez plusieurs `@`).

Exemple :

`alias1@@alias2@@@alias3`

Ceci exécutera alias1, puis attendra 2 secondes avant alias2, et 3 secondes avant alias3.

### `@github_issue@` - Créer un nouveau ticket dans le dépôt Lanceur-bis

Crée un nouveau ticket GitHub pour le projet Lanceur-bis. Le paramètre est le titre du ticket.

Pour utiliser cette fonctionnalité, vous devez disposer d'un jeton fourni par quelqu'un ayant accès administrateur au dépôt.

Exemple :

`<alias_issue> Bug : quelque chose ne fonctionne pas`

## Cartes de remplacement

Toute occurrence de ces macros dans le champ _Nom du fichier_ sera remplacée comme suit :

| Carte    | Description                                                       |
| -------- | ----------------------------------------------------------------- |
| **\$C$** | Remplacé par le contenu du presse-papiers, formaté pour les URL.  |
| **\$R$** | Remplacé par le contenu brut du presse-papiers.                   |
| **\$I$** | Remplacé par les paramètres saisis.                               |
| **\$W$** | Remplacé par les paramètres saisis, formatés pour les URL.        |
